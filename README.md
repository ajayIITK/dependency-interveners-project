# Dependency Interveners Project

This repository contains the implementation for the project:

"Structural Constraints on Intervening Nodes in Dependency Structures: A Cross-Linguistic Study"

## What this code does:
- Loads Universal Dependencies treebanks
- Extracts head–dependent relations
- Identifies intervening nodes
- Computes:
  - number of interveners
  - arity (number of dependents)
  - phrase length (subtree size)
- Performs analysis across:
  - dependency distance
  - sentence length
  - dependency relations
  - multiple languages
